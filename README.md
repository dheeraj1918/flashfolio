# flashfolio
FLASHFOLIO - This website helps you to make your portfolio website under 60s.

For reference use this website link: https://flashfolio1.netlify.app/

#Portfolio website generation process
Step 1: Navigate to the Home page or Generate Page.
Step 2: Click on "Get Started Now!" if you are in home page then upload your resume by browse it through your system  and select the style.
Step 3: wait for 30 - 40s for generate of a link.
Step 4: The generated link will be appears at the same page.
Step 5: The link activation may take 20-30s.
Step 6: If you dont like the website try again the process.
